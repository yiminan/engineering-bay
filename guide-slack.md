# guide-slack.md

## Slack

| Shortcut            | Description          |
|---------------------|----------------------|
| command + K         | 모든 검색                |
| shift + esc         | 모든 채널의 안읽은 메시지 모두 읽기 |
| esc                 | 해당 채널 안읽은 메시지 모두 읽기  |
| command + F         | 현재 채널에서 검색           |
| command + shift + F | 모든 채널에서 검색           |
| command + shift + T | 쓰레드 보기               |
